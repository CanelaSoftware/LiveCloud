# function cdb_ping()
---
## Summary:
Pings the server and returns the response time in milliseconds

## Inputs:
None

## Outputs:
(String) - the response time of the CDB server in milliseconds

## Additional Requirements:
This API call requires internet access.

## API Version:
* `0.3` - Introduced

## Examples:
```
put cdb_ping()

#Output: 359.479904
```
