# function cdb_sdkBuild()
---
## Summary
Used to see when your CDB libraries were built. Can be used to see if you are out of date.

## Inputs
None
	
## Outputs
* (String) - The DateTime that the libraries were built.

## Examples
```livecodeserver
put cdb_sdkBuild()
# Output: 2018-12-12 14:12:54.818
```
